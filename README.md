# Django Base Project

Базовый Django-проект с аутентификацией пользователей, расширенными профилями, нефтяными компаниями, скважинами и модулем суточных рапортов по добыче.

---

## Функционал

### 🏢 Компании (OilCompany)
- создание компаний
- редактирование компаний
- удаление компаний
- просмотр списка компаний
- связь с пользователями и скважинами
- фильтрация и сортировка

---

### 👤 Пользователи и профили
- регистрация пользователей
- вход и выход из системы
- автоматическое создание профиля
- CRUD пользователей
- редактирование профиля (включая других пользователей)
- поля профиля:
  - компания
  - отдел
  - телефон
  - bio
- просмотр списка пользователей
- фильтрация по компании
- сортировка
- пагинация
- доступ только для авторизованных пользователей
- управление через Django Admin

---

### 🛢 Скважины (Well)
- создание скважин
- редактирование скважин
- удаление скважин
- просмотр списка скважин
- поля:
  - название
  - компания
  - тип
  - максимальная глубина
  - координаты (latitude, longitude)
- фильтрация по компании
- сортировка
- пагинация
- управление через Django Admin

---

### 📊 Суточные рапорты по добыче (DailyProduction)
- создание рапортов
- редактирование рапортов
- удаление рапортов
- просмотр списка рапортов
- автоматический расчет чистого дебита нефти
- фильтрация:
  - по компании
  - по скважине
  - по диапазону дат
- сортировка
- пагинация
- защита страниц через авторизацию
- управление через Django Admin

## Установка и запуск проекта

### 1. Клонировать репозиторий
```bash
git clone https://github.com/USERNAME/REPOSITORY_NAME.git
cd REPOSITORY_NAME
```

### 2. Создать виртуальное окружение
```bash
python -m venv venv
```

### 3. Активировать виртуальное окружение

#### Windows
```bash
venv\Scripts\activate
```

#### macOS / Linux
```bash
source venv/bin/activate
```

### 4. Установить зависимости
```bash
pip install django
```

### 5. Миграции
```bash
python manage.py makemigrations
python manage.py migrate
```

### 6. Создать суперпользователя
```bash
python manage.py createsuperuser
```

### 7. Запуск сервера
```bash
python manage.py runserver
```

## Модели проекта

### Profile
Расширяет стандартную модель пользователя Django.

Поля:
- `bio`
- `department`
- `phone_number`
- `oil_company`

---

### OilCompany
Модель нефтяной компании.

Поля:
- `name`
- `region`

Связи:
- имеет пользователей
- имеет скважины

---

### Well
Модель скважины.

Поля:
- `name`
- `oil_company`
- `type`
- `max_drilling_depth`
- `latitude`
- `longitude`

---

### DailyProduction
Модель суточного рапорта.

Поля:
- `well`
- `date`
- `work_time`
- `liquid_debit`
- `water_cut`
- `oil_density`

---

## Бизнес-логика

Формула расчета нефти:

```text
Дебит нефти = Дебит жидкости × (1 - Обводненность / 100) × Плотность нефти
```

---

## Валидация

- все поля обязательны
- `work_time` — от 0 до 24
- `water_cut` — от 0 до 100
- уникальность: `(well, date)`

Все ошибки переведены на русский язык.
