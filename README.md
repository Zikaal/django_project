# Django Base Project

Базовый Django-проект с аутентификацией пользователей, расширенными профилями и административной панелью.

## Функционал

- регистрация пользователей
- вход и выход из системы
- автоматическое создание профиля для нового пользователя
- редактирование профиля
- просмотр списка всех пользователей
- доступ к защищенным страницам только для авторизованных пользователей
- управление профилями через Django Admin

## Используемые технологии

- Python
- Django
- SQLite3

## Установка и запуск проекта

### 1. Клонировать репозиторий
```bash
git clone https://github.com/USERNAME/REPOSITORY_NAME.git
cd REPOSITORY_NAME
```

### 2. Создать виртуальное окружение
```bash
python -m venv venv
```

### 3. Активировать виртуальное окружение

#### Windows
```bash
venv\Scripts\activate
```

#### macOS / Linux
```bash
source venv/bin/activate
```

### 4. Установить зависимости
```bash
pip install django
```

### 5. Применить миграции
```bash
python manage.py migrate
```

### 6. Создать суперпользователя
```bash
python manage.py createsuperuser
```

### 7. Запустить сервер
```bash
python manage.py runserver
```

После запуска проект будет доступен по адресу:

```bash
http://127.0.0.1:8000/
```

## Основные страницы

- Регистрация: `/accounts/register/`
- Вход: `/accounts/login/`
- Выход: `/accounts/logout/`
- Профиль: `/accounts/profile/`
- Список пользователей: `/accounts/users/`
- Админ-панель: `/admin/`

## Модель Profile

Для расширения стандартной модели пользователя Django была создана модель `Profile`, связанная с `User` через `OneToOneField`.

Поля профиля:
- `bio` — информация о пользователе
- `department` — отдел
- `phone_number` — номер телефона

При создании нового пользователя профиль создается автоматически через Django Signals (`post_save`).

## Доступ

Следующие страницы доступны только авторизованным пользователям:

- список пользователей
- страница профиля

## Админ-панель

Модель `Profile` зарегистрирована в Django Admin, поэтому профилями можно управлять через встроенную административную панель.

## Автор

Проект создан в учебных целях как базовый фундамент для веб-приложения на Django.
